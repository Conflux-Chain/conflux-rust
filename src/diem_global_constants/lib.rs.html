8//! The purpose of this crate is to offer a single source of truth for the
9//! definitions of shared constants within the Diem codebase. This is useful
10//! because many different components within Diem often require access to global
11//! constant definitions (e.g., Diem Safety Rules, the Key Manager, and Secure
12//! Storage). To avoid duplicating these definitions across crates (and better
13//! allow these constants to be updated in a single location), we define them
14//! here.
15#![forbid(unsafe_code)]
16
